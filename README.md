# DiceFun

### A simple dice game.

Played between two players. Players can enter their name and randomely dice will roll and the player who gets greater number will win. If numbers are equal then match draw.

Technology used:

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>
